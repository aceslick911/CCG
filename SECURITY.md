# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CCG, please report it responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, send an email to the maintainer's gmail (angeloperera) or use GitHub's private vulnerability reporting feature (if available).

We will acknowledge receipt of your report within 48 hours and aim to provide a fix or mitigation plan as quickly as possible.

## Supported Versions

Only the latest version on the main branch is supported with security updates.
