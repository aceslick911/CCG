CCG — Privacy Policy

Last updated: 6 July 2026

CCG does not collect, store, or transmit your data. Anywhere. Ever.

CCG (exCel Code Generator) runs entirely inside your Excel client. Formula parsing, analysis, and code generation all happen locally in the add-in's task pane. Your workbook contents, formulas, generated code, and cell values never leave your machine via CCG.

Specifics

  • No analytics, no telemetry, no tracking — the add-in contains no analytics code of any kind.
  • No accounts, no sign-in — CCG has no backend service.
  • Static hosting — the task pane's files are served from GitHub Pages. GitHub may record standard web-server logs (IP address, user agent) when the pane loads, per GitHub's privacy statement.
  • Office.js — the Microsoft Office JavaScript library is loaded from Microsoft's CDN, as required for all Office add-ins, per Microsoft's privacy statement.
  • Data written to your workbook — features you explicitly invoke (such as creating LAMBDA named functions) write into the workbook you're editing, and nowhere else.

Open source

CCG's complete source code is public at github.com/aceslick911/CCG — you can verify every claim above.
